Sobre mim

Meu nome é Lucas Barros Simon, nasci em Santos, São Paulo, no dia 21 de fevereiro de 2003. Atualmente, sou estudante universitário, cursando Sistemas de Informação na Universidade Santa Cecília (UNISANTA). Minha paixão por programação começou cedo. Quando criança, costumava criar plugins de Minecraft em Java e sempre tive curiosidade por explorar novas tecnologias. Esse interesse cresceu ao longo dos anos, levando-me a aprofundar em áreas como desenvolvimento web, análise de dados e criação de soluções inovadoras.
Além da programação, sou apaixonado por design, especialmente por temas relacionados à moda e estética, e me inspiro em figuras como Jerry Lorenzo, KAWS e Kid Cudi. No meu tempo livre, gosto de assistir a filmes e séries, e frequentemente faço análises críticas de obras cinematográficas. Estou sempre em busca de novos desafios e oportunidades para crescer e contribuir com minhas habilidades e conhecimentos.

Meus Projetos

Cinema Database

O Cinema Database é um projeto web desenvolvido inteiramente em HTML, CSS e JavaScript, oferecendo uma experiência personalizada de busca por filmes e diretores. Esta aplicação permite aos usuários explorar e encontrar informações sobre obras cinematográficas de forma dinâmica e intuitiva.

Utilizando JavaScript para manipulação de DOM e integração com dados, o Cinema Database processa as pesquisas dos usuários em tempo real, exibindo resultados relevantes diretamente na tela. A aplicação foi construída com foco na usabilidade e na experiência do usuário.

Além disso, o projeto está em constante desenvolvimento, com planos de expandir a base de dados e adicionar novas funcionalidades, garantindo uma experiência cada vez mais rica para os amantes de cinema.

Calculadora em Java

Desenvolvi uma calculadora em Java que oferece tanto funcionalidades básicas, como adição, subtração, multiplicação e divisão, quanto recursos avançados, incluindo cálculo de porcentagens e operações com parênteses. A interface foi criada usando Swing, proporcionando uma experiência de usuário intuitiva e responsiva, com botões organizados de maneira clara e funcional.

Além das operações tradicionais, a calculadora também lida com exceções, como divisão por zero, garantindo maior robustez e confiabilidade. O projeto é modular e segue boas práticas de programação orientada a objetos, facilitando a manutenção e a adição de novas funcionalidades, como operações científicas ou um histórico de cálculos.
